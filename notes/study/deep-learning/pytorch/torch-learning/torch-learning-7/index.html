PyTorch еӯҰд№  -7- и®ӯз»ғе’ҢиҜ„дј°
Yiwei Zhang 2023е№ҙ7жңҲ20ж—Ҙ дёӢеҚҲ
3k еӯ— 25 еҲҶй’ҹ 0 ж¬Ў   0 дәә   0 жқЎ

жң¬ж–ҮжңҖеҗҺжӣҙж–°дәҺпјҡ2024е№ҙ1жңҲ14ж—Ҙ жҷҡдёҠ

еңЁеүҚж–ҮеҹәзЎҖдёҠпјҢжҲ‘д»¬е·Із»ҸиҺ·еҫ—дәҶж•°жҚ®гҖҒеј йҮҸе’ҢжҚҹеӨұеҮҪж•°пјҢ жң¬ж–Үд»Ӣз»Қ Pytorch зҡ„иҝӣиЎҢи®ӯз»ғе’ҢиҜ„дј°зҡ„ж ёеҝғжөҒзЁӢ гҖӮ

еҸӮиҖғ ж·ұе…Ҙжө…еҮәPyTorch пјҢзі»з»ҹиЎҘйҪҗеҹәзЎҖзҹҘиҜҶгҖӮ

жң¬иҠӮзӣ®еҪ•

  • PyTorchзҡ„и®ӯз»ғ/иҜ„дј°жЁЎејҸзҡ„ејҖеҗҜ
  • е®Ңж•ҙзҡ„и®ӯз»ғ/иҜ„дј°жөҒзЁӢ

жЁЎеһӢжЁЎејҸ

йҰ–е…Ҳеә”иҜҘи®ҫзҪ®жЁЎеһӢзҡ„зҠ¶жҖҒпјҡеҰӮжһңжҳҜи®ӯз»ғзҠ¶жҖҒпјҢйӮЈд№ҲжЁЎеһӢзҡ„еҸӮж•°еә”иҜҘж”ҜжҢҒеҸҚеҗ‘дј ж’ӯзҡ„дҝ®ж”№пјӣеҰӮжһңжҳҜйӘҢиҜҒ/жөӢиҜ•зҠ¶жҖҒпјҢеҲҷдёҚеә”иҜҘдҝ®ж”№жЁЎеһӢеҸӮж•°гҖӮеңЁPyTorchдёӯпјҢжЁЎеһӢзҡ„зҠ¶жҖҒи®ҫзҪ®йқһеёёз®ҖдҫҝпјҢеҰӮдёӢзҡ„дёӨдёӘж“ҚдҪңдәҢйҖүдёҖеҚіеҸҜпјҡ

model.train()

model.train() зҡ„дҪңз”ЁжҳҜеҗҜз”Ё Batch Normalization е’Ң DropoutгҖӮ

еҰӮжһңжЁЎеһӢдёӯжңүBNеұӮ(Batch Normalizationпјүе’ҢDropoutпјҢйңҖиҰҒеңЁи®ӯз»ғж—¶ж·»еҠ  model.train()гҖӮmodel.train()жҳҜдҝқиҜҒBNеұӮиғҪеӨҹз”ЁеҲ°жҜҸдёҖжү№ж•°жҚ®зҡ„еқҮеҖје’Ңж–№е·®гҖӮеҜ№дәҺDropoutпјҢmodel.train()жҳҜйҡҸжңәеҸ–дёҖйғЁеҲҶзҪ‘з»ңиҝһжҺҘжқҘи®ӯз»ғжӣҙж–°еҸӮж•°гҖӮ

model.eval()

model.eval()зҡ„дҪңз”ЁжҳҜдёҚеҗҜз”Ё Batch Normalization е’Ң DropoutгҖӮ

еҰӮжһңжЁЎеһӢдёӯжңүBNеұӮ(Batch Normalizationпјүе’ҢDropoutпјҢеңЁжөӢиҜ•ж—¶ж·»еҠ model.eval()гҖӮmodel.eval()жҳҜдҝқиҜҒBNеұӮиғҪеӨҹз”Ёе…ЁйғЁи®ӯз»ғж•°жҚ®зҡ„еқҮеҖје’Ңж–№е·®пјҢеҚіжөӢиҜ•иҝҮзЁӢдёӯиҰҒдҝқиҜҒBNеұӮзҡ„еқҮеҖје’Ңж–№е·®дёҚеҸҳгҖӮеҜ№дәҺDropoutпјҢmodel.eval()жҳҜеҲ©з”ЁеҲ°дәҶжүҖжңүзҪ‘з»ңиҝһжҺҘпјҢеҚідёҚиҝӣиЎҢйҡҸжңәиҲҚејғзҘһз»Ҹе…ғгҖӮ

и®ӯз»ғе®Ңtrainж ·жң¬еҗҺпјҢз”ҹжҲҗзҡ„жЁЎеһӢmodelиҰҒз”ЁжқҘжөӢиҜ•ж ·жң¬гҖӮеңЁmodel(test)д№ӢеүҚпјҢйңҖиҰҒеҠ дёҠmodel.eval()пјҢеҗҰеҲҷзҡ„иҜқпјҢжңүиҫ“е…Ҙж•°жҚ®пјҢеҚідҪҝдёҚи®ӯз»ғпјҢе®ғд№ҹдјҡж”№еҸҳжқғеҖјгҖӮиҝҷжҳҜmodelдёӯеҗ«жңүBNеұӮе’ҢDropoutжүҖеёҰжқҘзҡ„зҡ„жҖ§иҙЁгҖӮ

и®ӯз»ғжөҒзЁӢ

ж•°жҚ®еҠ иҪҪ

жҲ‘д»¬еүҚйқўеңЁDataLoaderжһ„е»әе®ҢжҲҗеҗҺд»Ӣз»ҚдәҶеҰӮдҪ•д»ҺдёӯиҜ»еҸ–ж•°жҚ®пјҢеңЁи®ӯз»ғиҝҮзЁӢдёӯдҪҝз”Ёзұ»дјјзҡ„ж“ҚдҪңеҚіеҸҜпјҢеҢәеҲ«еңЁдәҺжӯӨж—¶иҰҒз”ЁforеҫӘзҺҜиҜ»еҸ–DataLoaderдёӯзҡ„е…ЁйғЁж•°жҚ®гҖӮ

д№ӢеҗҺе°Ҷж•°жҚ®ж”ҫеҲ°GPUдёҠз”ЁдәҺеҗҺз»ӯи®Ўз®—пјҢжӯӨеӨ„д»Ҙ.cuda()дёәдҫӢ

zero_grad

ејҖе§Ӣз”ЁеҪ“еүҚжү№ж¬Ўж•°жҚ®еҒҡи®ӯз»ғж—¶пјҢеә”еҪ“е…Ҳе°ҶдјҳеҢ–еҷЁзҡ„жўҜеәҰзҪ®йӣ¶пјҡ

еҮҪж•°дјҡйҒҚеҺҶжЁЎеһӢзҡ„жүҖжңүеҸӮж•°пјҢйҖҡиҝҮеҶ…зҪ®ж–№жі•жҲӘж–ӯеҸҚеҗ‘дј ж’ӯзҡ„жўҜеәҰжөҒпјҢеҶҚе°ҶжҜҸдёӘеҸӮж•°зҡ„жўҜеәҰеҖји®ҫдёә0пјҢеҚідёҠдёҖж¬Ўзҡ„жўҜеәҰи®°еҪ•иў«жё…з©әгҖӮ

loss

д№ӢеҗҺе°ҶdataйҖҒе…ҘжЁЎеһӢдёӯи®ӯз»ғпјҡ

ж №жҚ®йў„е…Ҳе®ҡд№үзҡ„criterionи®Ўз®—жҚҹеӨұеҮҪж•°пјҡ

backward

е°ҶlossеҸҚеҗ‘дј ж’ӯеӣһзҪ‘з»ңпјҡ

PyTorchзҡ„еҸҚеҗ‘дј ж’ӯ(еҚіtensor.backward())жҳҜйҖҡиҝҮautogradеҢ…жқҘе®һзҺ°зҡ„пјҢautogradеҢ…дјҡж №жҚ®tensorиҝӣиЎҢиҝҮзҡ„ж•°еӯҰиҝҗз®—жқҘиҮӘеҠЁи®Ўз®—е…¶еҜ№еә”зҡ„жўҜеәҰгҖӮ

е…·дҪ“жқҘиҜҙпјҢtorch.tensorжҳҜautogradеҢ…зҡ„еҹәзЎҖзұ»пјҢеҰӮжһңдҪ и®ҫзҪ®tensorзҡ„requires_gradsдёәTrueпјҢе°ұдјҡејҖе§Ӣи·ҹиёӘиҝҷдёӘtensorдёҠйқўзҡ„жүҖжңүиҝҗз®—пјҢеҰӮжһңдҪ еҒҡе®Ңиҝҗз®—еҗҺдҪҝз”Ёtensor.backward()пјҢжүҖжңүзҡ„жўҜеәҰе°ұдјҡиҮӘеҠЁиҝҗз®—пјҢtensorзҡ„жўҜеәҰе°ҶдјҡзҙҜеҠ еҲ°е®ғзҡ„.gradеұһжҖ§йҮҢйқўеҺ»гҖӮ

жӣҙе…·дҪ“ең°иҜҙпјҢжҚҹеӨұеҮҪж•°lossжҳҜз”ұжЁЎеһӢзҡ„жүҖжңүжқғйҮҚwз»ҸиҝҮдёҖзі»еҲ—иҝҗз®—еҫ—еҲ°зҡ„пјҢиӢҘжҹҗдёӘwзҡ„requires_gradsдёәTrueпјҢеҲҷwзҡ„жүҖжңүдёҠеұӮеҸӮж•°пјҲеҗҺйқўеұӮзҡ„жқғйҮҚwпјүзҡ„.grad_fnеұһжҖ§дёӯе°ұдҝқеӯҳдәҶеҜ№еә”зҡ„иҝҗз®—пјҢз„¶еҗҺеңЁдҪҝз”Ёloss.backward()еҗҺпјҢдјҡдёҖеұӮеұӮзҡ„еҸҚеҗ‘дј ж’ӯи®Ўз®—жҜҸдёӘwзҡ„жўҜеәҰеҖјпјҢе№¶дҝқеӯҳеҲ°иҜҘwзҡ„.gradеұһжҖ§дёӯгҖӮ

еҰӮжһңжІЎжңүиҝӣиЎҢtensor.backward()зҡ„иҜқпјҢжўҜеәҰеҖје°ҶдјҡжҳҜNoneпјҢеӣ жӯӨloss.backward()иҰҒеҶҷеңЁoptimizer.step()д№ӢеүҚгҖӮ

жӣҙж–°еҸӮж•°

дҪҝз”ЁдјҳеҢ–еҷЁжӣҙж–°жЁЎеһӢеҸӮж•°пјҡ

step()еҮҪж•°зҡ„дҪңз”ЁжҳҜжү§иЎҢдёҖж¬ЎдјҳеҢ–жӯҘйӘӨпјҢйҖҡиҝҮжўҜеәҰдёӢйҷҚжі•жқҘжӣҙж–°еҸӮж•°зҡ„еҖјгҖӮеӣ дёәжўҜеәҰдёӢйҷҚжҳҜеҹәдәҺжўҜеәҰзҡ„пјҢжүҖд»ҘеңЁжү§иЎҢoptimizer.step()еҮҪж•°еүҚеә”е…Ҳжү§иЎҢloss.backward()еҮҪж•°жқҘи®Ўз®—жўҜеәҰгҖӮ

жіЁж„ҸпјҡoptimizerеҸӘиҙҹиҙЈйҖҡиҝҮжўҜеәҰдёӢйҷҚиҝӣиЎҢдјҳеҢ–пјҢиҖҢдёҚиҙҹиҙЈдә§з”ҹжўҜеәҰпјҢжўҜеәҰжҳҜtensor.backward()ж–№жі•дә§з”ҹзҡ„гҖӮ

йӘҢиҜҒжөҒзЁӢ

йӘҢиҜҒ/жөӢиҜ•зҡ„жөҒзЁӢеҹәжң¬дёҺи®ӯз»ғиҝҮзЁӢдёҖиҮҙпјҢдёҚеҗҢзӮ№еңЁдәҺпјҡ

  • йңҖиҰҒйў„е…Ҳи®ҫзҪ®torch.no_gradпјҢд»ҘеҸҠе°Ҷmodelи°ғиҮіevalжЁЎејҸ
  • дёҚйңҖиҰҒе°ҶдјҳеҢ–еҷЁзҡ„жўҜеәҰзҪ®йӣ¶
  • дёҚйңҖиҰҒе°ҶlossеҸҚеҗ‘еӣһдј еҲ°зҪ‘з»ң
  • дёҚйңҖиҰҒжӣҙж–°optimizer

зӨәдҫӢд»Јз Ғ

дёҖдёӘе®Ңж•ҙзҡ„еӣҫеғҸеҲҶзұ»зҡ„и®ӯз»ғиҝҮзЁӢеҰӮдёӢжүҖзӨәпјҡ

еҜ№еә”зҡ„пјҢдёҖдёӘе®Ңж•ҙеӣҫеғҸеҲҶзұ»зҡ„йӘҢиҜҒиҝҮзЁӢеҰӮдёӢжүҖзӨәпјҡ
