Python CUDA зј–зЁӢ - 6 - е…ұдә«еҶ…еӯҳ
Yiwei Zhang 2021е№ҙ4жңҲ21ж—Ҙ еҮҢжҷЁ
5.6k еӯ— 47 еҲҶй’ҹ 0 ж¬Ў   0 дәә   0 жқЎ

жң¬ж–ҮжңҖеҗҺжӣҙж–°дәҺпјҡ2023е№ҙ8жңҲ8ж—Ҙ дёӯеҚҲ

CUDAзј–зЁӢдёӯеҶ…еӯҳеҲҶдёәдё»жңәеҶ…еӯҳпјҲеҶ…еӯҳжқЎпјүдёҺи®ҫеӨҮеҶ…еӯҳпјҲжҳҫеӯҳпјүпјҢдёәжҸҗй«ҳи®Ўз®—ж•ҲзҺҮпјҢйңҖиҰҒи®ҫи®ЎзЁӢеәҸйҷҚдҪҺеҶ…еӯҳзҡ„ж•°жҚ®жҗ¬иҝҗпјҢжҲ–дҪҝз”Ёеҝ«йҖҹзҡ„еҶ…еӯҳеҜ„еӯҳж•°жҚ®гҖӮ

е…ұдә«еҶ…еӯҳ

CPUе’ҢGPUз»„жҲҗејӮжһ„и®Ўз®—жһ¶жһ„пјҢеҰӮжһңжғід»ҺеҶ…еӯҳдёҠдјҳеҢ–зЁӢеәҸпјҢжҲ‘д»¬еҝ…йЎ»е°ҪйҮҸеҮҸе°‘дё»жңәдёҺGPUи®ҫеӨҮй—ҙзҡ„ж•°жҚ®жӢ·иҙқпјҢе№¶е°ҶжӣҙеӨҡи®Ўз®—д»Һдё»жңәз«ҜиҪ¬з§»еҲ°GPUи®ҫеӨҮз«ҜпјҢжҲ‘д»¬иҰҒе°ҪйҮҸеңЁи®ҫеӨҮз«ҜеҲқе§ӢеҢ–ж•°жҚ®пјҢе№¶и®Ўз®—дёӯй—ҙж•°жҚ®пјҢе№¶е°ҪйҮҸдёҚеҒҡж— ж„Ҹд№үзҡ„ж•°жҚ®еӣһеҶҷгҖӮ

GPUзҡ„еҶ…еӯҳз»“жһ„еҰӮеӣҫжүҖзӨәпјҡGPUзҡ„и®Ўз®—ж ёеҝғйғҪеңЁStreaming MultiprocessorпјҲSMпјүдёҠпјҢSMйҮҢжңүи®Ўз®—ж ёеҝғеҸҜзӣҙжҺҘи®ҝй—®зҡ„еҜ„еӯҳеҷЁпјҲRegisterпјүе’Ңе…ұдә«еҶ…еӯҳпјҲShared MemoryпјүпјӣеӨҡдёӘSMеҸҜд»ҘиҜ»еҸ–жҳҫеҚЎдёҠзҡ„жҳҫеӯҳпјҢеҢ…жӢ¬е…ЁеұҖеҶ…еӯҳпјҲGlobal MemoryпјүгҖӮ

жҜҸдёӘSMдёҠзҡ„Shared MemoryзӣёеҪ“дәҺиҜҘSMдёҠзҡ„дёҖдёӘзј“еӯҳпјҢдёҖиҲ¬йғҪеҫҲе°ҸпјҢTelsa V100зҡ„Shared Memoryд№ҹеҸӘжңү96KBгҖӮжіЁж„ҸпјҢShared Memoryе’ҢGlobal Memoryзҡ„еӯ—йқўдёҠйғҪжңүе…ұдә«зҡ„ж„ҸжҖқпјҢдҪҶжҳҜдёҚиҰҒе°ҶдёӨиҖ…зҡ„жҰӮеҝөж··ж·ҶпјҢShared MemoryзҰ»и®Ўз®—ж ёеҝғжӣҙиҝ‘пјҢе»¶иҝҹеҫҲдҪҺпјӣGlobal MemoryжҳҜж•ҙдёӘжҳҫеҚЎдёҠзҡ„е…ЁеұҖеҶ…еӯҳпјҢе»¶иҝҹй«ҳгҖӮ

д»ҺиҪҜд»¶и§’еәҰжқҘзңӢпјҢCUDAзҡ„зәҝзЁӢеҸҜд»Ҙи®ҝй—®дёҚеҗҢзә§еҲ«зҡ„еӯҳеӮЁпјҢжҜҸдёӘThreadжңүзӢ¬з«Ӣзҡ„з§ҒжңүеҶ…еӯҳпјӣжҜҸдёӘBlockдёӯеӨҡдёӘThreadйғҪеҸҜд»ҘеңЁиҜҘBlockзҡ„Shared MemoryдёӯиҜ»еҶҷж•°жҚ®пјӣж•ҙдёӘGridдёӯжүҖжңүThreadйғҪеҸҜд»ҘиҜ»еҶҷGlobal MemoryгҖӮShared Memoryзҡ„иҜ»еҶҷи®ҝй—®йҖҹеәҰдјҡиҝңй«ҳдәҺGlobal MemoryгҖӮеҶ…еӯҳдјҳеҢ–дёҖиҲ¬дё»иҰҒеҲ©з”ЁShared MemoryжҠҖжңҜгҖӮдёӢж–Үе°Ҷд»Ҙзҹ©йҳөд№ҳжі•дёәдҫӢпјҢеұ•зӨәеҰӮдҪ•дҪҝз”ЁShared MemoryжқҘдјҳеҢ–зЁӢеәҸгҖӮ

жҷ®йҖҡзҹ©йҳөд№ҳжі•

дёҖдёӘC = ABзҡ„зҹ©йҳөд№ҳжі•иҝҗз®—пјҢйңҖиҰҒжҲ‘д»¬жҠҠAзҡ„жҹҗдёҖиЎҢдёҺBзҡ„жҹҗдёҖеҲ—зҡ„жүҖжңүе…ғзҙ дёҖдёҖзӣёд№ҳпјҢжұӮе’ҢеҗҺпјҢе°Ҷз»“жһңеӯҳеӮЁеҲ°з»“жһңзҹ©йҳөCзҡ„(row, col)дёҠгҖӮеңЁиҝҷз§Қе®һзҺ°дёӯпјҢжҜҸдёӘзәҝзЁӢйғҪиҰҒиҜ»еҸ–Aзҡ„дёҖж•ҙиЎҢе’ҢBзҡ„дёҖж•ҙеҲ—пјҢе…ұи®Ўз®—MиЎҢ*PеҲ—гҖӮд»Ҙи®Ўз®—з¬¬rowиЎҢдёәдҫӢпјҢи®Ўз®—C[row, 0]гҖҒC[row, 1]вҖҰC[row, p-1]иҝҷдәӣзӮ№ж—¶йғҪйңҖиҰҒд»Һжҳҫеӯҳзҡ„Global MemoryдёӯжҠҠж•ҙдёӘз¬¬rowиЎҢиҜ»еҸ–дёҖйҒҚгҖӮеҸҜд»Ҙз®—еҲ°пјҢAзҹ©йҳөдёӯзҡ„жҜҸдёӘзӮ№йңҖиҰҒиў«иҜ» B.width ж¬ЎпјҢBзҹ©йҳөдёӯзҡ„жҜҸдёӘзӮ№йңҖиҰҒиў«иҜ» A.height ж¬ЎгҖӮиҝҷж ·жҜ”иҫғжөӘиҙ№ж—¶й—ҙгҖӮеӣ жӯӨпјҢеҸҜд»Ҙе°ҶеӨҡж¬Ўи®ҝй—®зҡ„ж•°жҚ®ж”ҫеҲ°Shared MemoryдёӯпјҢеҮҸе°‘йҮҚеӨҚиҜ»еҸ–зҡ„ж¬Ўж•°пјҢе№¶е……еҲҶеҲ©з”ЁShared Memoryзҡ„е»¶иҝҹдҪҺзҡ„дјҳеҠҝгҖӮ

еҹәдәҺShared Memoryзҡ„зҹ©йҳөд№ҳжі•

жҺҘдёӢжқҘзҡ„зЁӢеәҸеҲ©з”ЁдәҶShared MemoryжқҘеҒҡзҹ©йҳөд№ҳжі•гҖӮиҝҷдёӘе®һзҺ°дёӯпјҢи·ҹжңӘеҒҡдјҳеҢ–зҡ„зүҲжң¬зӣёеҗҢзҡ„жҳҜпјҢжҜҸдёӘThreadи®Ўз®—з»“жһңзҹ©йҳөдёӯзҡ„дёҖдёӘе…ғзҙ пјҢдёҚеҗҢзҡ„жҳҜпјҢжҜҸдёӘCUDA Blockдјҡд»ҘдёҖдёӘ BLOCK_SIZE * BLOCK_SIZE еӯҗзҹ©йҳөдёәеҹәжң¬зҡ„и®Ўз®—еҚ•е…ғгҖӮе…·дҪ“иҖҢиЁҖпјҢйңҖиҰҒеЈ°жҳҺShared MemoryеҢәеҹҹпјҢж•°жҚ®з¬¬дёҖж¬Ўдјҡд»ҺGlobal MemoryжӢ·иҙқеҲ°Shared MemoryдёҠпјҢжҺҘдёӢжқҘеҸҜеӨҡж¬ЎйҮҚеӨҚеҲ©з”ЁShared MemoryдёҠзҡ„ж•°жҚ®гҖӮ

иҝӣиЎҢShared MemoryдјҳеҢ–еҗҺпјҢи®Ўз®—йғЁеҲҶзҡ„иҖ—ж—¶еҮҸе°‘дәҶиҝ‘дёҖеҚҠпјҡ

иЎҘе……иҜҙжҳҺ

  1. еЈ°жҳҺShared MemoryгҖӮиҝҷйҮҢдҪҝз”ЁдәҶcuda.shared.array(shape,type)пјҢshapeдёәиҝҷеқ—ж•°жҚ®зҡ„еҗ‘йҮҸз»ҙеәҰеӨ§е°ҸпјҢtypeдёәNumbaж•°жҚ®зұ»еһӢпјҢдҫӢеҰӮжҳҜint32иҝҳжҳҜfloat32гҖӮиҝҷдёӘеҮҪж•°еҸӘиғҪеңЁи®ҫеӨҮз«ҜдҪҝз”ЁгҖӮе®ҡд№үеҘҪеҗҺпјҢиҝҷеқ—ж•°жҚ®еҸҜиў«еҗҢдёҖдёӘBlockзҡ„жүҖжңүThreadе…ұдә«гҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢиҝҷеқ—ж•°жҚ®иҷҪз„¶еңЁж ёеҮҪж•°дёӯе®ҡд№үпјҢдҪҶе®ғдёҚжҳҜеҚ•дёӘThreadзҡ„з§Ғжңүж•°жҚ®пјҢе®ғеҸҜиў«еҗҢBlockдёӯзҡ„жүҖжңүThreadиҜ»еҶҷгҖӮ
  2. ж•°жҚ®еҠ иҪҪгҖӮжҜҸдёӘThreadдјҡе°ҶAдёӯзҡ„дёҖдёӘе…ғзҙ еҠ иҪҪеҲ°sAдёӯпјҢдёҖдёӘBlockзҡ„ BLOCK_SIZE x BLOCK_SIZE дёӘThreadеҸҜд»ҘжҠҠsAеЎ«е……ж»ЎгҖӮcuda.syncthreads()дјҡзӯүеҫ…BlockдёӯжүҖжңүThreadжү§иЎҢе®Ңд№ӢеҗҺжүҚжү§иЎҢдёӢдёҖжӯҘгҖӮжүҖд»ҘпјҢеҪ“жү§иЎҢе®ҢиҝҷдёӘеҮҪж•°зҡ„ж—¶еҖҷпјҢsAе’ҢsBзҡ„ж•°жҚ®е·Із»ҸжӢ·иҙқеҘҪдәҶгҖӮ
  3. ж•°жҚ®еӨҚз”ЁгҖӮAдёӯзҡ„жҹҗдёӘзӮ№пјҢеҸӘдјҡиў«иҜ»еҸ– B.width / BLOCK_SIZE ж¬ЎпјӣBдёӯзҡ„жҹҗдёӘзӮ№пјҢеҸӘдјҡиў«иҜ» A.height / BLOCK_SIZE ж¬ЎгҖӮfor n in range(BLOCK_SIZE)иҝҷдёӘеҫӘзҺҜеҒҡеӯҗзҹ©йҳөеҗ‘йҮҸд№ҳжі•ж—¶пјҢеҸҜеӨҡж¬ЎеӨҚз”ЁsAе’ҢsBзҡ„ж•°жҚ®гҖӮ
  4. еӯҗзҹ©йҳөзҡ„ж•°жҚ®жұҮжҖ»гҖӮжҲ‘д»¬д»ҘдёҖдёӘ BLOCK_SIZE x BLOCK_SIZE зҡ„еӯҗзҹ©йҳөдёәеҚ•дҪҚеҲҶеҲ«еҜ№Aд»Һе·ҰеҲ°еҸіпјҢеҜ№Bд»ҺдёҠеҲ°дёӢе№із§»е№¶и®Ўз®—пјҢе…ұеҫӘзҺҜ A.width / BLOCK_SIZE ж¬ЎгҖӮеңЁжҹҗдёҖжӯҘе№із§»пјҢдјҡеҫ—еҲ°еӯҗзҹ©йҳөзҡ„зӮ№з§ҜгҖӮfor m in range(math.ceil(A.shape[1] / BLOCK_SIZE))иҝҷдёӘеҫӘзҺҜиө·еҲ°дәҶи®Ўз®—Aд»Һе·ҰеҲ°еҸідёҺBд»ҺдёҠеҲ°дёӢзӮ№з§Ҝзҡ„иҝҮзЁӢгҖӮ
