简介

OpenClaw 是一个可扩展的 AI 助手框架，核心特性：

  • 记忆系统：持久化存储重要信息，支持语义搜索
  • 技能系统：将工作流程打包成可复用模块

常见问题

Q: 记忆搜索返回空结果？

  1. 检查 MEMORY.md 是否存在
  2. 确认 memorySearch.provider 配置正确
  3. 验证 node-llama-cpp 依赖已安装
