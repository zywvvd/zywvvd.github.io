工作学习中经常会查阅各种网站的资料，或者使用很多在线工具，如果可以在自己的博客中创建个人的站点导航应该会方便得多，本文记录实现过程。

实现思路

主要的实现流程图如下

graph TD
	B[导航图标] -- 入口 --> C[主题 layout 页] 
    D[主题配置文件] -- 读取导航信息 --> C
    E[后端截屏服务] 
    C -- 请求截屏地址 --> E
    E -- 生成 --> F((截屏图像))
    F -- 使用 --> C
    G[特效样式]
    G -- 应用 -->C
    H((CSS)) --> G
    I((JS))-->G
    style C fill:#3DD,stroke:#333,stroke-width:0px

主题 layout 文件

我的主题是 fluid，在 themes\fluid\layout 中创建 site-map.ejs 文件

核心目的：

  1. 获取主题配置文件中的站点信息
  2. 整理站点信息向后端请求截屏服务
  3. 根据站点信息创建网页 html 文件
