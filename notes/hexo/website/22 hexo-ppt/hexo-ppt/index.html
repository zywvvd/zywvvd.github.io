本文最后更新于：2021年9月12日 下午

PPT 是一种流行的信息展现方式，nodeppt 又是网页版 PPT 一股清流，本文记录nodeppt 制作流程以及在Hexo博客展示的方法。

使用教程

在Hexo中展示PPT

  • 编辑自己的 nodePPT
  • 导出 html
  • 生成 dist 文件夹，其中包含了展示需要的文件
  • 将这部分生成的文件夹放在 source 文件夹中，但是需要配置 skip_render 不要渲染这部分文件
    • hexo -> source -> nodeppt -> ... 同时在主题配置文件 _config.yml 中配置：
