亲手搭建台式主机记录
Yiwei Zhang 2022年8月7日 晚上
1.2k 字 10 分钟 0 次   0 人   0 条

本文最后更新于：2024年5月7日 下午

之前的笔记本用了七年多已经不堪重负，狠心整个台式机，本文记录新台式机配置过程。

配置思路

性能需求

  • CPU、显卡中上等水平，最起码使用起来不能卡顿，可以训练神经网络模型
  • 主板满足CPU 显卡需求， 支持 DDR5内存，性能稳定
  • 电源稳定，功率足够用
  • 内存要够大，可以流畅使用
  • 固态硬盘装系统和对速度要求较高的软件
  • 机械硬盘装数据
  • 支持 wifi6 无线网卡
  • 风扇要酷 ~

稳定性

  • 自己攒的机器一定要保证兼容性和鲁棒性
  • 纠结了一阵子选择主要在华硕家买硬件，毕竟稳定老字号了
  • 硬盘、内存选择金士顿
  • CPU 本就稳定，而且买不到假货，散片就好
  • 散热要好，准备上360水冷
  • 电源大功率总是好的

主板

  • 7代的 cpu 应该也够日常使用了，没打算超频，放弃了 Z690
  • 在重炮手和迫击炮之间纠结了一下，虽然没有迫击炮接口丰富，但还是选择了华硕这个牌子
  • DDR5 毕竟速度快些，未来应该是趋势，提前占坑应该没啥大问题

CPU

  • i7 12700 足够足够用，没考虑 i9, 考虑到万一显卡坏了咋办，没有买 12700F，稳还是很重要的
  • CPU 没有假货，买拼多多散片也几乎不会上当
  • 至于体质好坏，无所谓了，个人足够用

安装过程

第一次安装CPU，记录一下安装过程
1. CPU 插槽 (注意左下角三角)
2. 掀开插槽盖子
3. CPU 三角对应盖子上的三角
4. 盖上盖子(过程中不能弹回）

显卡

  • 本想入 3060 12G的大显存卡的，但是后来发现3060除了显存大没什么太大优势了
  • 相比来说加500块上 3060Ti 足够用了
  • B660 的 PCIE 5x 16 应该能插未来很多年的显卡了，实在有需要还可以换卡

电源

  • 考虑到华硕的主板应该更喜欢自己家的电源吧
  • 现在的功率在 300W 左右，750W的电源应该足够用了

内存

  • DDR5
  • 金士顿.
  • 16G × 2

固态硬盘

  • 金士顿
  • 1 T
  • M.2 接口

机械硬盘

  • 这个希捷 6T 的硬盘我买第 7 块了，虽然便宜但是用了一年没出什么问题
  • 而且价格从 680 降到了现在的 430

机箱

  • 爱国者（aigo）YOGO K1 钛灰色
  • 这个机箱真的可以吹爆，设计合理，功能齐全，天衣无缝，几乎完美

CPU 散热

  • 听大佬推荐的 利民冰封幻境 就上车了
  • 虽然不打算超频但是散热好点总没坏处吧
  • 第一次使用水冷，用起来好像除了价格贵别的都挺好的

无线模块

  • B660 重炮手自带了 wifi6 天线，速度、信号都还可以
  • 断过一次，拔了重新插好了
  • 不过也有槽点，这个外置网卡没地方放啊，躺在机箱上很不优雅
  • 于是就钉在机箱上的风扇螺丝上了

机箱风扇

  • 拼多多的 利民12cm 风扇
  • 风量虽然不算大，但是我的机箱可以在装完360水冷后还能装6个风扇
  • 和水冷的风扇颜色一样，很漂亮
  • 声音几乎没有，安静好看又便宜

显示器

  • 显示器是此次选购的一点瑕疵了
  • 分辨率很高，刷新率也不低，但也就仅此而已
  • 颜色渐变没有十分柔和、暗色、白色区分不够极致，漏光严重，还有些刺眼，只能一直开着夜晚模式
  • 不过也没什么可说的，毕竟一千多的显示器，努力今后改善吧

音箱

  • 随便买的音箱，听不出什么音质
  • 但是足够酷炫

鲁大师 6.10 跑分
