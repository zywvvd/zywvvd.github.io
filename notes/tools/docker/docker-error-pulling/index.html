将 Docker 的默认镜像源换成国内的镜像源。国内镜像源通常响应速度更快，更加稳定。

自 2024-06-06 开始，国内的 Docker Hub 镜像加速器相继停止服务，可选择为 Docker daemon 配置代理或自建镜像加速服务。
