Typora 的国产开源替代产品 Marktext
Yiwei Zhang 2022年3月4日 上午
472 字 4 分钟 0 次   0 人   0 条

本文最后更新于：2024年5月11日 下午

Typora 好用，但是眼下收费又闭源，使用中一直有bug没有解决；刚刚发现了国产替代产品MarkText，尽管存在很多问题，但设计的功能几乎可以覆盖 Typora，并且添加了很多个性化配置和新功能，本文并介绍相关功能。

简介

A simple and elegant open-source markdown editor that focused on speed and usability. —— MarkText

MarkText 是一个 MIT 授权的开源项目，最新版本总是可以从 GitHub 发布页面免费下载，是一款国产的开源 Markdown 文本编辑器。

目前仍在开发中，问题还是存在的，希望可以逐步改进，开源软件我们也可以做些贡献。

  • 官网链接：https://marktext.app/

  • 官方 Github：https://github.com/marktext/marktext

安装使用

  • 下载地址：https://marktext.app/

  • 下载对应安装包安装适用即可

相对于Typora的新功能

  • 粘贴链接自动转换为引用格式

  • 可以编辑图像，自动转为 html 代码

  • 支持多种主题

  • 可配置界面

  • 支持方便贴心的小按钮

    • 方便地编辑文字
    • 段落编辑按钮
  • 支持很多快捷键，虽然很多不好使

快捷键

功能 快捷键 备注
X级标题 ctrl + X X∈[1~6] （貌似不好使）
加粗 Ctrl + B 默认加粗
倾斜 Ctrl + I
源码模式 Ctrl + E
新建文档 Ctrl + T
插入表格 Ctrl + Shift + T
侧边文件信息显示 Ctrl + J
删除线 Ctrl + D 删除的文字
导航栏 Ctrl + K
搜索 Ctrl + F
超链接 Ctrl + L
插入公式 Ctrl+M
分割线 ctrl + alt + -
调整字体与行间距 Ctrl+>
插入图片 Ctrl+shift+I
插入表情包 Ctrl+/

当前问题

  • 大部分快捷键不好使
  • 无法直接粘贴表格
