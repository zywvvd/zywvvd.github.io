Git - 配置全局 ignore

本文最后更新于：2021年7月24日 晚上

使用 git 工具管理代码时，经常会误把临时文件夹、IDE临时文件当作代码的更新给出新文件提示，每个工程配置一个ignore文件又太繁琐，本文记录全局ignore方法。

解决方案

  1. 创建.gitignore_global 文件

Win

  • 在 C盘 -> 用户 -> 当前用户 -> 创建 .gitignore_global 文件

Linux

  • 在家目录创建 ~/.gitignore_global 文件

  1. 添加忽略内容

  • 向 .gitignore_global文件加入需要忽略的内容

  1. 执行git指令
